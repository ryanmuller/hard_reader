Advanced usage of Hard Reader

There really isn't anything special to using Hard Reader itself, but there are countless ways you can use email to create your own experience. Let me know if you have other examples!

Automatically send a link

If you're already saving links to a bookmarking service or program like Evernote, you don't need to take an extra step to send it to Hard Reader. You can use IFTTT to automatically send email to read@mg.learnstream.org. Just make sure the email is sent from the address you used to sign up to Hard Reader.

Multiple accounts

Don't want all of your summaries to appear in the same review? Maybe you have one personal review and one you like to send to colleagues. All you need to do is sign up for multiple accounts on Hard Reader. You don't even need to create a bunch of email addresses: Some email providers, such as Gmail, will allow you to create any address for you@example.com in the form you+whatever@example.com. Follow these instructions to make these different addresses available in Gmail when sending to Hard Reader. That's all there is to it: any notes sent from a particular address will be grouped into its account.

Read later

If you're trying to keep your inbox clean, but you have too many Hard Reader notes to respond to, you can use Boomerang to save them for another day.
